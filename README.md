# eslint-plugin-no-do-expression

no do expression

## Installation

```
$ npm install 
```


## Usage

Add `no-do-expression` to the plugins section of your `.eslintrc` configuration file. You can omit the `eslint-plugin-` prefix:

```json
{
    "plugins": [
        "no-do-expression"
    ]
}
```
